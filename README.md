# StepUp

Small actions. Real change. Every day.

## What it is
StepUp is an accountability app for New Year goals (and any goals). Set goals, post daily check-ins, join groups, and support others through consistent action.

## Core Features (MVP)
- Authentication + Profiles
- Goals (daily/weekly cadence)
- Daily Check-ins (streaks + history)
- Groups (invite + group feed)
- Comments + reactions
- Reminders

## Tech Stack
- Next.js (React)
- TypeScript
- PostgreSQL (Supabase)
- Supabase Auth
- Vercel (deployment)
